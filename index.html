About This Project

These chapters were built as an interactive learning journey through Data Vault 2.0 and its surrounding ecosystem. Each one is a standalone visual experience with exercises, quizzes, and concept explanations using the 32nd-century programmable matter aesthetic.

Topics span from foundational ontology (what IS a Hub?) through SQL construction, dbt tooling, EU regulatory compliance (GDPR, DORA, AI Act), and Python for data engineering. The chapters use Skyrim, Star Trek, and The Matrix as analogy domains to make abstract architecture tangible.
